## 一、什麼是機器學習？

**一句話**：讓電腦從數據中學習規律，而不是人手寫規則。

### 三大範式

| 類型 | 定義 | 例子 |
|------|------|------|
| **監督學習** | 有標籤數據，學習輸入→輸出的映射 | 圖像分類、房價預測 |
| **非監督學習** | 無標籤數據，發現數據內在結構 | 聚類、降維 |
| **強化學習** | 通過與環境互動學習最優策略 | 遊戲AI、機器人控制 |

NOAI 主要考**監督學習**和**非監督學習**。

## 二、數學基礎（你必須知道的）

### 線性代數

| 概念 | 用途 | 一句話解釋 |
|------|------|-----------|
| 向量 | 數據表示 | 一組數字，如 [1, 2, 3] |
| 矩陣 | 數據批量處理 | 二維數組，如 [[1,2],[3,4]] |
| 矩陣乘法 | 神經網絡核心運算 | 行×列，結果矩陣的每個元素是一組內積 |
| 轉置 | 矩陣行列互換 | A[i][j] → A[j][i] |
| 特徵值/特徵向量 | PCA 降維的核心 | Av = λv，v 是方向，λ 是縮放比例 |
| SVD | 矩陣分解、推薦系統 | 矩陣 = U × Σ × V^T |

### 概率與統計

| 概念 | 用途 | 一句話解釋 |
|------|------|-----------|
| 條件概率 P(A\|B) | 貝葉斯分類器 | 在 B 發生的前提下 A 發生的概率 |
| 貝葉斯定理 | 樸素貝葉斯 | P(A\|B) = P(B\|A)×P(A) / P(B) |
| 期望 E[X] | 損失函數 | 隨機變量的平均值 |
| 方差 Var(X) | 模型評估 | 數據偏離均值的程度 |
| 正態分佈 | 數據假設 | 鐘形曲線，大部分數據集中在均值附近 |
| 交叉熵 | 分類損失函數 | 衡量兩個概率分佈的差異 |

### 微積分

| 概念 | 用途 | 一句話解釋 |
|------|------|-----------|
| 導數 | 理解梯度 | 函數在某點的變化率 |
| 偏導數 | 多參數優化 | 對某一個變量求導，其他固定 |
| 梯度 | 反向傳播 | 所有偏導數組成的向量，指向函數增長最快的方向 |
| 鏈式法則 | 反向傳播的數學基礎 | dy/dx = dy/du × du/dx |
| 梯度下降 | 模型訓練核心 | 沿梯度反方向更新參數，使損失函數減小 |

## 三、核心 ML 算法

### 監督學習

#### 線性回歸
- **做什麼**：預測連續值（如房價）
- **原理**：y = wx + b，找最佳的 w 和 b
- **損失**：均方誤差 MSE = (1/n)Σ(y - ŷ)²
- **優化**：梯度下降

#### 邏輯回歸
- **做什麼**：二分類（如垃圾郵件判斷）
- **原理**：在線性回歸基礎上加 Sigmoid 函數壓縮到 [0,1]
- **損失**：交叉熵
- **注意**：雖然叫"回歸"，但實際是分類算法

#### 決策樹
- **做什麼**：分類和回歸
- **原理**：通過一系列 if-else 條件分裂數據
- **關鍵指標**：信息增益、基尼係數
- **優點**：可解釋性強
- **缺點**：容易過擬合

#### 隨機森林
- **做什麼**：決策樹的集成版本
- **原理**：訓練多棵決策樹，投票/平均
- **優點**：比單棵樹更穩定、更準確
- **關鍵**：Bagging（隨機抽樣訓練每棵樹）

#### SVM（支持向量機）
- **做什麼**：分類
- **原理**：找到最大間隔的分隔超平面
- **核技巧**：處理非線性數據（RBF 核、多項式核）
- **適用**：小數據集、高維特徵

#### XGBoost / LightGBM
- **做什麼**：比賽神器，結構化數據首選
- **原理**：Boosting（逐步訓練，每棵樹修正前一棵的錯誤）
- **優點**：效果好、速度快
- **Kaggle 必備**

### 非監督學習

#### K-Means 聚類
- **做什麼**：把數據分成 K 組
- **原理**：隨機初始化中心點 → 分配 → 更新中心 → 重複
- **關鍵**：選擇 K 值（肘部法則）

#### PCA（主成分分析）
- **做什麼**：降維
- **原理**：找到數據方差最大的方向，投影
- **步驟**：標準化 → 協方差矩陣 → 特徵值分解 → 選前 k 個特徵向量

## 四、模型評估

### 分類指標

| 指標 | 公式 | 含義 |
|------|------|------|
| **Accuracy** | (TP+TN) / 全部 | 整體正確率 |
| **Precision** | TP / (TP+FP) | 預測為正的裡面，真正為正的比例 |
| **Recall** | TP / (TP+FN) | 真正為正的裡面，被正確預測的比例 |
| **F1 Score** | 2×P×R / (P+R) | Precision 和 Recall 的調和平均 |
| **ROC-AUC** | 曲線下面積 | 模型區分正負樣本的能力 |

### 回歸指標

| 指標 | 公式 | 含義 |
|------|------|------|
| **MSE** | (1/n)Σ(y-ŷ)² | 均方誤差 |
| **RMSE** | √MSE | 均方根誤差 |
| **MAE** | (1/n)Σ\|y-ŷ\| | 平均絕對誤差 |
| **R²** | 1 - SS_res/SS_tot | 模型解釋的方差比例 |

### 過擬合 vs 欠擬合

| 問題 | 表現 | 原因 | 解決方法 |
|------|------|------|----------|
| **過擬合** | 訓練好，測試差 | 模型太複雜，記住了噪聲 | 正則化、Dropout、更多數據、Early Stopping |
| **欠擬合** | 訓練和測試都差 | 模型太簡單 | 增加模型複雜度、更多特徵 |

### 交叉驗證
- **K-Fold**：數據分 K 份，輪流用 1 份做驗證、K-1 份做訓練
- **目的**：更可靠地估計模型性能，避免單次劃分的隨機性

## 五、特徵工程

### 數值特徵
- **標準化**：(x - mean) / std → 均值0，方差1
- **歸一化**：(x - min) / (max - min) → 壓縮到 [0,1]
- **對數變換**：處理偏態分佈
- **多項式特徵**：特徵之間的交互項

### 類別特徵
- **One-Hot 編碼**：類別 → 向量（如 紅=[1,0,0], 綠=[0,1,0], 藍=[0,0,1]）
- **Label 編碼**：類別 → 數字（如 紅=0, 綠=1, 藍=2）
- **Target 編碼**：用目標變量的均值替換類別

### 缺失值處理
- 刪除行/列
- 均值/中位數/眾數填充
- 模型預測填充（如 KNN）

### 特徵選擇
- 過濾法：相關性、方差閾值
- 包裹法：前向/後向選擇
- 嵌入法：L1 正則化、樹模型特徵重要性

## 六、深度學習基礎

### 神經網絡基本結構

```
輸入層 → 隱藏層1 → 隱藏層2 → ... → 輸出層
```

每一層的計算：`output = activation(W × input + b)`

### 激活函數

| 函數 | 公式 | 特點 |
|------|------|------|
| **ReLU** | max(0, x) | 最常用，計算快，解決梯度消失 |
| **Sigmoid** | 1/(1+e^(-x)) | 輸出 [0,1]，用於二分類輸出層 |
| **Softmax** | e^x_i / Σe^x_j | 多分類輸出層，輸出概率分佈 |
| **Tanh** | (e^x-e^(-x))/(e^x+e^(-x)) | 輸出 [-1,1]，比 Sigmoid 好 |

### 反向傳播
1. 前向傳播：計算預測值
2. 計算損失：預測值 vs 真實值
3. 反向傳播：用鏈式法則計算梯度
4. 更新參數：w = w - lr × gradient

### 優化器

| 優化器 | 特點 |
|--------|------|
| **SGD** | 隨機梯度下降，基礎版 |
| **Momentum** | 加動量，收斂更快 |
| **Adam** | 自適應學習率，最常用 |
| **AdamW** | Adam + 權重衰減，訓練更穩定 |

### 學習率調度
- **StepLR**：每隔 N 個 epoch 衰減
- **CosineAnnealing**：餘弦退火
- **ReduceOnPlateau**：驗證損失不下降時衰減

## 七、經典網絡架構

### CNN（卷積神經網絡）— 圖像

```
輸入圖像 → [卷積 → 激活 → 池化] × N → 全連接層 → 輸出
```

- **卷積層**：提取局部特徵（邊緣、紋理、形狀）
- **池化層**：降低維度（MaxPooling, AveragePooling）
- **經典模型**：LeNet → AlexNet → VGG → ResNet → EfficientNet

### RNN / LSTM — 序列

- **RNN**：處理序列數據，但有梯度消失問題
- **LSTM**：加了門控機制（遺忘門、輸入門、輸出門），解決長距離依賴
- **應用**：文本分類、時間序列預測

### Transformer — NLP 和更多

- **核心**：Self-Attention 機制
- **優點**：並行計算、捕捉長距離依賴
- **關鍵概念**：Query, Key, Value; Multi-Head Attention
- **代表模型**：BERT（理解）、GPT（生成）

## 八、PyTorch 核心概念

### Tensor
```python
import torch
x = torch.tensor([1.0, 2.0, 3.0])  # 創建
y = torch.randn(3, 4)               # 隨機 3x4 矩陣
z = x @ y                           # 矩陣運算
```

### nn.Module（模型定義）
```python
import torch.nn as nn

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 10)
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.relu(self.fc1(x))
        x = self.fc2(x)
        return x
```

### 訓練循環
```python
model = MyModel()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()

for epoch in range(num_epochs):
    for batch_x, batch_y in dataloader:
        pred = model(batch_x)           # 前向
        loss = criterion(pred, batch_y) # 損失
        optimizer.zero_grad()           # 清零梯度
        loss.backward()                 # 反向傳播
        optimizer.step()                # 更新參數
```

### DataLoader
```python
from torch.utils.data import DataLoader, TensorDataset

dataset = TensorDataset(X_train, y_train)
dataloader = DataLoader(dataset, batch_size=32, shuffle=True)
```
