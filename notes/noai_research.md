## 一、NOAI 是什麼？

**NOAI**（National Olympiad in Artificial Intelligence）= **全國人工智能奧林匹克**

- **定位**：IOAI（國際人工智能奧林匹克）在中國的**國家隊選拔賽**
- **目標**：每年選出 **8 名學生**組成中國國家隊，代表中國參加 IOAI 國際賽
- **歷史**：已舉辦兩屆；從第三屆起只面向**高中生**
- **性質**：公益導向，非營利
- **學術指導單位**：北京中關村學院
- **白金贊助**：淘天集團（覆蓋國家隊選拔、AI訓練營、集訓和出征）
- **官方測試平台**：深勢科技**玻爾平台**（Bohrium）

## 三、學術大綱核心領域

根據官方描述，NOAI 考核方向為：

> "以機器學習、深度學習為主，以數據驅動為主要特征，屬於較為專業的人工智能領域知識技能，而非泛人工智能。"

具體涵蓋：

### A. 數學基礎
- 線性代數（矩陣運算、特徵值分解、SVD）
- 概率論與統計（貝葉斯定理、分佈、假設檢驗）
- 微積分（梯度、鏈式法則、優化）
- 信息論基礎（熵、交叉熵、KL散度）

### B. Python 編程與算法
- Python 基礎語法與數據結構
- NumPy / Pandas 數據處理
- 常見算法（排序、搜索、圖論基礎）
- 數據預處理與特徵工程

### C. 機器學習
- 監督學習（線性回歸、邏輯回歸、SVM、決策樹、隨機森林、XGBoost）
- 非監督學習（聚類、降維、PCA）
- 模型評估（交叉驗證、ROC/AUC、過擬合/欠擬合）
- 超參數調優
- 集成學習

### D. 深度學習
- 神經網絡基礎（前饋、反向傳播）
- CNN（卷積神經網絡）— 計算機視覺
- RNN / LSTM / GRU — 序列數據
- Transformer / Attention 機制 — NLP
- 預訓練模型（BERT、GPT 系列）
- PyTorch 實戰
- 模型訓練技巧（學習率調度、正則化、數據增強）

## 四、IOAI 國際賽概況

### 基本信息
- **全稱**：International Olympiad in Artificial Intelligence
- **第三屆**：2026年8月2–8日，哈薩克斯坦阿斯塔納
- **第四屆**：2027年，新加坡
- **參賽國**：40+ 個國家/地區
- **競賽平台**：Bohrium（玻爾平台）
- **網絡限制**：僅允許訪問文檔網站和集成的 LLM

### 競賽形式
- 包含 **At-home practice round**（賽前約一個月發布，不計入成績）
- 正式比賽包含理論和實操部分
- 任務類型：**寫代碼、在訓練數據上擬合模型、在測試數據上運行推理**
- 涵蓋領域：機器學習、NLP、計算機視覺

### 各國 NOAI 賽制參考

| 國家 | 賽制特點 |
|------|----------|
| 中國 | 兩輪（紙筆+上機），選8人 |
| 美國/加拿大 | 三輪（含線下邀請輪+選拔營） |
| 日本 | 以 Kaggle 競賽為起點，含面試 |
| 巴西 | 71.6萬參賽者中選4人 |
| 俄羅斯 | 多階段，含訓練→資格→正式→決賽 |
| 新加坡 | 300道MCQ/3小時初賽 → 決賽 |
| 香港 | 800+人 → 25人 → 12人 → 8人 |

## 六、備考策略總結

### 第一輪（水平測試）備考重點
1. **Python 編程**：熟練掌握語法、數據結構、NumPy/Pandas
2. **數學基礎**：線性代數、概率統計、微積分
3. **AI 基本概念**：監督/非監督學習、常見算法原理
4. **刷題**：MCQ + 編程題，限時練習

### 第二輪（中國站）備考重點
1. **PyTorch 實戰**：熟練使用 PyTorch 構建和訓練模型
2. **Kaggle 練習**：參加 Kaggle 競賽積累實戰經驗
3. **模型調優**：學習率、正則化、數據增強等技巧
4. **閱讀論文**：了解 SOTA 方法
5. **熟悉玻爾平台**：提前了解官方競賽平台的使用

## 七、黑歷史與吐槽

> ⚠️ 以下內容來自公開討論，真實性待考

### 爭議點
1. **認可度問題**：NOAI 相比 NOI（信息學奧賽）歷史較短，部分人對其含金量存疑
2. **商業化傾向**：有聲音質疑賽事是否過於商業化（企業贊助、收費培訓等）
3. **公平性**：雲端算力競賽是否對資源不足的學生不公平
4. **與 NOI 的關係**：AI 方向與傳統算法競賽的定位差異
5. **俄羅斯賽區醜聞**：977 名學生因抄襲被取消資格（IOAI 層面）

### 正面評價
1. 填補了 AI 領域青少年競賽的空白
2. 與國際接軌（IOAI 體系）
3. 注重實操能力，而非純理論
4. 提供了進入 AI 領域的競技平台
