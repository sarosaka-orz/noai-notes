# NOAI Notes

My personal notes, solutions, and study materials for the NOAI (National Olympiad in AI) competition.

## What is NOAI?

NOAI（全國人工智能奧林匹克）是 IOAI（國際人工智能奧林匹克）在中國的國家隊選拔賽，每年選出 8 名學生代表中國參加國際賽。
